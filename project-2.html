Probability of default of credit card clients

Led AI-based credit risk assessment project, predicting default probability for credit card clients. Utilized machine learning algorithms, data preprocessing, and feature engineering to analyze historical data. Achieved 85% accuracy in identifying defaulters. Collaborated with cross-functional teams to deploy the model for real-time decision-making.

Code Link

Project Overview

Objective
The primary objective of this study is to develop data-driven predictive models to identify credit card defaulters. The dataset includes various attributes such as limit balance, age, sex, amount of bill statement, repayment status, and amount of previous payment. The aim is to determine the strongest predictors of default and to predict which customers are likely to default on their credit card payments.
Dataset
The study utilizes the UCI dataset, which contains information about credit card clients and their historical transactions. Key features include demographic details, credit limits, and payment behaviors.
Methodology
Multiple machine learning models are trained and evaluated to predict credit card defaults. The models employed in this study include:
Logistic Regression
K-Nearest Neighbors (KNN)
Decision Tree
Na√Øve Bayes
Random Forest
Methods
These models are trained using repeated cross-validation and evaluated on a testing set. The study also applies data preprocessing techniques such as Principal Component Analysis (PCA) to handle dataset imbalance issues through oversampling and undersampling.
Tools and Environment
The analysis is conducted using Python, leveraging several machine learning and statistical libraries:
scikit-learn: For implementing machine learning algorithms
numpy and pandas: For data manipulation and analysis
imblearn: For handling imbalanced datasets
matplotlib and seaborn: For data visualization

Results
The base model achieved an R-square value of 81% using all variables without any preprocessing. Data preprocessing, including PCA, slightly improved the model performance and reduced computational cost. Oversampling techniques performed marginally better than undersampling due to training on a larger dataset. All implemented models yielded comparable accuracy results.
Conclusion
The study successfully identifies key drivers of credit card default and demonstrates the effectiveness of various supervised learning algorithms in predicting default. Data preprocessing and balancing techniques enhance model performance, although all models show similar accuracy. The findings highlight the potential of machine learning in financial risk management, particularly in automating the prediction of credit card defaults.
